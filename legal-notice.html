Mentions légales

Erwann Jault

Étudiant de troisième année à l'ECV Digital Bordeaux, j'ai développé ce site web dans le cadre d'un projet visant à expérimenter avec le langage web CSS.

Navigation

La navigation a été faite en utilisant une combinaison de Flexbox et display : inline-block. Le menu est en position : sticky pour permettre d'accéder au menu en tout temps. Grâce à l'utilisation de media queries (et d'un peu de Javascript), ce dernier se transforme en burger menu et la navigation, en utilisant flex-direction : column, passe d'horizontal à vertical.

Accueil

L'accueil a été développé en utilisant une première section de CSS Columns, se dégradant de 3 à 1 colonne(s) prenant une largeur maximale de 320 pixels grâce à columns : 3 320px. La section Personnalités est structuré à l'aide d'un conteneur Grid CSS créant trois colonnes. Ce grand conteneur est divisé en autant de conteneurs Flexbox que de personnalités. Ces conteneurs permettent d'agencer de façon modulaire l'image par rapport au texte. La hero Image a été fait en donnant une hauteur avec la portion de code height : 60vh et en utilisant les propriétés CSS Background. En utilisant les propriétés de la variable webfont Bitter et les propriétés CSS Transitions, les titres changent leur graisse au survol.

Propriétés

La page Propriétés utilise les attributs CSS de Flexbox et de transformation. Les éléments sont en ligne grâce à flex-direction : row, sont espacés les uns des autres d'un même espace grâce à justify-content : space-between et sont centrés sur l'axe verticale du conteneur principale grâce à align-items : center. Le code flex-wrap : no-wrap contraint les éléments à loger sur une seule et même ligne. Sur le titre, un transform : rotate(90deg) est appliquée.

Personnalités

La page Personnalités utilise Grid CSS et Flexbox ensemble. La page est séparée en deux colonnes grâce à grid-template-column : repeat(2,1fr) et chaque section est structurée visuellement par Flexbox avec un conteneur principal avec un flex-direction : row; et deux conteneurs secondaires en flex-direction : column. L'image utilise object-fit : cover avec une hauteur, une largeur et un border-radius donnés.

Jen Simmons

La page de présentation de Jen Simmons met à profit les Grid CSS, les propriétés de l'élément HTML Picture ainsi que Object-fit et max-width. En utilisant les Media Queries, le contenu entier se dégrade de deux à une colonne quand l'écran rétrécit. On notera aussi que les paragraphes de l'entièreté du site sont contraints à une max-width : 440px, ce qui correspond à une ligne de texte entre 55 à 65 caractères soit la longueur optimale pour lire.
